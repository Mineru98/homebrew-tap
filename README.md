# homebrew-tap

Homebrew tap for Mineru98's macOS apps.

## Installation

### Screen Cleaner

A macOS utility that blocks keyboard input and dims the screen while you clean your MacBook.

```bash
brew tap mineru98/tap
brew install --cask screen-cleaner
```

## Apps

| App | Description |
|-----|-------------|
| [screen-cleaner](https://github.com/Mineru98/screen-cleaner) | Blocks keyboard input and dims screen while cleaning your MacBook |
